## Descricao

Back-End do projeto de uma plataforma de movimentação de servidores.

Sendo desenvolvido com:

- NestJS
- Prisma ORM
- TypeScript
- PostgreSQL

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
